Stone RealEstate — modular contained portfolio

This folder contains a modularized real estate portfolio (Stone RealEstate) extracted from an original HTML file.

Structure:
- `index.html` — entry file referencing local CSS/JS
- `css/styles.css` — extracted styles
- `js/app.js` — small UI helpers (smooth scroll + input focus)

Access: http://localhost:3000/stone-realestate

To add more contained portfolios, create new folders under `sites/` and add a static route in `server.js`.
